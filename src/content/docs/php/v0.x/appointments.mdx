# Appointments

`$client->appointments` books and cancels appointments. All methods require a bearer
token. The server resolves the slot from `doctorId` + `appointmentDate` — there is
no `slotId` in the request.

| Method                                                  | Endpoint |
|---------------------------------------------------------|----------|
| `reserveInterview(doctorId, appointmentDate, appointmentType?)` | `POST /patients/addInterviewDateReservation` |
| `addPhysical(doctorId, appointmentDate)`                | `POST /patients/addNewAppointment` |
| `cancel(eventId)`                                       | `DELETE /patients/deleteUserAppointment/{eventId}` |

## `reserveInterview(...)`

Reserve an online (interview) slot. Returns `null` on success.

```php
$client->appointments->reserveInterview(
    doctorId: 8282,
    appointmentDate: '2026-06-20 14:30', // "Y-m-d H:i", today..+21
);
```

`reserveInterview` parameters:

| Parameter         | Type          | Required | Default |
|-------------------|---------------|:--------:|---------|
| `doctorId`        | `int\|string` | ✓        | — |
| `appointmentDate` | `string` (`Y-m-d H:i`) | ✓ | — |
| `appointmentType` | `string` (`interview` \| `appointment`) | — | `'interview'` |

A successful reservation returns `null` (the API responds `resultType 0`, `data: null`).
Conflicts and validation failures throw a typed [exception](/docs/php/v0.x/errors/).

## `addPhysical(...)`

Create an in-person (physical) appointment. No `appointmentType`.

```php
$client->appointments->addPhysical(
    doctorId: 8282,
    appointmentDate: '2026-06-20 14:30', // "Y-m-d H:i"
);
```

Both parameters are required: `doctorId` (`int|string`) and `appointmentDate` (`string`).

## `cancel(eventId)`

Cancel an appointment by its event id (`cln_events.id`).

```php
$client->appointments->cancel($eventId);
```

`eventId` is typed `int|string`.

> Cancellation can be rejected (server-side) for insurance appointments, once the
> cancel window has passed, or if the event is not found — these surface as a typed
> exception.
